How it Works?

1. ReSplitter filter's with Regex and assign's an ID to every line in every file/folder that you selected and save's them in cache.

2. Knowing the ID the program is able to replace all the coincidences on all the files on any subfolder/folder.

3. The user is able to directly send the prompt to an AI and translate a single server/plugin in one prompt.

ReSplitter have's an Extension & API mode that allows you to translate for free all the ID's by parts to prevent being limited
